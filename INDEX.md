## 💡 Dicas de Uso

### Tip 1: Use DevTools (F12)

```
F12 → Console → Type: ThemeManager.setTheme('dark')
F12 → Application → Local Storage → Ver dados persistidos
F12 → Lighthouse → Check performance & accessibility
```

### Tip 2: Customize Facilmente

Cores estão em CSS (linhas 11-35):
```css
:root {
    --color-primary: #2563eb;  ← Mude para sua cor!
    --color-secondary: #1e40af;
}
```

Copie um `.portfolio-card` em `index.html` seção `#portfolio`
Mude título, descrição, tecnologias
Recarregue navegador - pronto!
