## 🚀 Como Usar

### 1. Executar Localmente

```bash
# Python
cd projeto_portifolio
python -m http.server 8000
# Acesse: http://localhost:8000

# Ou abra index.html no navegador
open index.html
```

### 2. Testar Funcionalidades

- ✅ Clique no botão 🌙 para alternar tema
- ✅ Preencha formulário para testar validação
- ✅ Redimensione navegador para testar responsividade
- ✅ Pressione F12 para abrir DevTools

### 3. Customizar

1. Abra `index.html` - Edite conteúdo pessoal
2. Abra `style.css` - Customize cores/fontes
3. Abra `script.js` - Adicione funcionalidades
4. Recarregue página (Ctrl+F5)
